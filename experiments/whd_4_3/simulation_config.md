# 计划 4.3 节仿真配置

## 科学问题

固定发射端、接收端和环境，只改变人体—手机距离 `d_t`，是否会产生可观测的
复数 CSI 变化？

## 固定参数

| 参数             |                            数值 |
| ---------------- | ------------------------------: |
| 载波频率         |                         2.4 GHz |
| 带宽             |                          20 MHz |
| 子载波           |                 64 个均匀采样点 |
| 手机/发射端      |           `(0.0, 0.0, 1.2)` m |
| 主接收端         |           `(4.0, 0.0, 1.2)` m |
| 参考距离 `d_0` |                          0.40 m |
| 扫描范围 `d_t` |      0.25–0.70 m，间隔 0.025 m |
| 人体代理尺寸     |      `0.20 × 0.45 × 1.70` m |
| 人体材料参数     | `eps_r=40`、`sigma_e=1 S/m` |

手机和接收端保持不动，仅改变人体代理的位置。

## 仿真场景

1. **未遮挡 LOS 对照组**：人体沿链路垂直方向移动，并始终位于直射线之外。
   在仅包含直射路径的模型中，CSI 理论上不应发生变化。
2. **LOS 遮挡切换场景**：人体沿与链路夹角 38° 的方向移动。在
   `d_t≈0.37 m` 附近，人体长方体在“与直射线相交”和“不相交”之间切换。
   WiTwin 原生计算用于检查路径可见性；解析代理在被遮挡时增加两条人体边缘
   绕射项。
3. **人体邻近散射代理**：人体沿链路垂直方向移动。CSI 是直射项与一条
   人体相关散射项的相干叠加。散射强度扫描 `0.10`、`0.25` 和 `0.50`，其中
   `0.25` 为名义曲线。

第三种场景是几何光学敏感性代理，不是全波近场求解。它用于判断：如果存在
一条随人体移动的相干次级路径，`d_t` 是否可能变得可观测，以及该结论对假定
散射强度的依赖程度。

## 指标和操作性阈值

所有残差均以 `d_0=0.40 m` 对应的 CSI 为参考。

- 幅度残差：各子载波 `20 log10(|H_k|)` 差值的均方根。
- 相位残差：各子载波环绕相位差的均方根，单位为度。
- 复数 NRMSE：`||H(d)-H(d_0)||_2 / ||H(d_0)||_2`。

在本次筛选实验中，满足以下任一条件即称为“可观测”：

- 幅度均方根残差不小于 `1 dB`；
- 相位均方根残差不小于 `10°`。

这些是明确设定的工程筛选阈值，不代表某一 CSI 采集设备的实际噪声下限。
真实数据验证必须使用硬件重复测量得到的波动范围替代这些阈值。

## 可复现性边界

- WiTwin 原生结果使用 `max_bounces=0`、`max_diffraction_order=0`，并在 CFR
  中保留物理绝对时延，即 `normalize_delays=False`。
- 脚本会记录软件包版本、源码版本、原始数组、表格指标和一次反射能力探针。
- 除非反射探针成功，否则当前固定组合不能用于声称反射路径 EPC 已通过验证。
