# `w_geo` 长轨迹并行 D0--D5 实验配置

## 1. 实验问题与设计原则

本实验回答：在房间、材料、接收端和真实运动轨迹不变时，不同类型的手机--人体三维几何误差会怎样改变复 CSI，并污染有效反射参数 `theta_ref` 的校准？

记 `t=0,...,79` 为离散时刻，`p_t` 为手机世界坐标位置，`R_t` 为手机姿态，`q_t=(p_t,R_t)` 为手机六自由度位姿；`r_t` 是从手机天线参考点指向人体盒体中心的真值三维向量，`d_t=||r_t||_2` 是手机--人体距离。人体中心为 `p_t+r_t`。

与旧实验不同，D0--D5 不再是轨迹的六个时间片段，而是施加在每个 `t` 上的六个并行几何条件：

```text
同一真实长轨迹 q_t、r_t
           │
           ├── theta_ref*=1 ──> 公共无噪声 CSI 真值 ──> 公共配对 CSI 噪声
           │
           ├── D0 几何 ──> 预测路径基
           ├── D1 几何 ──> 预测路径基
           ├── D2 几何 ──> 预测路径基
           ├── D3 几何 ──> 预测路径基
           ├── D4 几何 ──> 预测路径基
           └── D5 几何 ──> 预测路径基
```

因此同一时刻的六个条件拥有相同的房间、手机位姿、人体真值、AP、材料、CSI真值和CSI噪声，条件间只改变预测端使用的几何。

## 2. 房间、人体、AP与无线配置

房间世界坐标范围为：

```text
x ∈ [-3.0, 3.0] m
y ∈ [-2.0, 2.0] m
z ∈ [ 0.0, 3.0] m
```

房间由4面厚0.10 m的墙、地面和天花板构成。墙体材料为 `eps_r=4.0, sigma_e=0.01 S/m`；地面为 `eps_r=5.0, sigma_e=0.02 S/m`；天花板为 `eps_r=3.0, sigma_e=0.005 S/m`。

人体代理是始终平行世界坐标轴的盒体，尺寸为 `(0.22,0.30,1.70) m`，材料为 `eps_r=38.0, sigma_e=1.0 S/m`。实验只移动盒体中心，不模拟四肢、人体旋转、衣物或体型变化。

固定AP同时是CSI接收端，位置为：

```text
AP = (2.45, 1.55, 1.25) m
```

无线与路径配置如下：

| 项目 | 固定值 |
|---|---:|
| 载频 | 2.4 GHz |
| 带宽 | 20 MHz |
| 子载波 | 64个，2.39--2.41 GHz等间隔 |
| Tx/Rx | 单天线单元、dipole方向图、垂直极化 |
| 最大镜面反射 | `max_bounces=3` |
| 绕射 | `max_diffraction_order=0` |
| 主求解采样数 | `num_samples=2048` |
| 单场景路径上限 | `max_num_paths=192` |
| 路径贡献门限 | 0 |
| 反射后端 | `reflection_field_backend="drjit"` |

## 3. 80时刻长轨迹

轨迹由5个空间区块组成，每区块16个时刻。令区块内归一化索引 `u=i/15`：

| 区块 | 时刻 | 手机位置 `(x,y,z)`/m | 名义运动方向 |
|---|---:|---|---:|
| Z0-south | 0--15 | `(-2.20+4.40u, -1.20+0.06sin(2πu), 1.22+0.04sin(πu))` | 0° |
| Z1-east | 16--31 | `(2.20+0.06sin(2πu), -1.20+2.40u, 1.25+0.05sin(πu))` | 90° |
| Z2-north | 32--47 | `(2.20-4.40u, 1.20+0.06sin(2πu), 1.28+0.04sin(πu))` | 180° |
| Z3-west | 48--63 | `(-2.20+0.06sin(2πu), 1.20-2.40u, 1.24+0.05sin(πu))` | -90° |
| Z4-center | 64--79 | `(-1.85+3.70u, -0.85+1.70u+0.10sin(2πu), 1.18+0.08sin(πu))` | `atan2(1.70,3.70)` |

令全轨迹相位 `phi=4πt/79`，手机欧拉角依次为 yaw、pitch、roll：

```text
yaw(t)   = heading(t) + 14 sin(phi) degrees
pitch(t) = 6 sin(1.5 phi) degrees
roll(t)  = 5 cos(phi) degrees
```

真实手机--人体相对几何为：

```text
d_t         = 0.43 + 0.065 sin(phi) + 0.025 cos(2.5 phi) m
azimuth_t   = heading(t) + 90 + 24 sin(0.75 phi) degrees
elevation_t = 4 sin(1.25 phi) degrees

r_x = d cos(elevation) cos(azimuth)
r_y = d cos(elevation) sin(azimuth)
r_z = d sin(elevation)
```

实际覆盖审计显示：手机在x/y/z方向分别跨越 `4.519/2.519/0.140 m`，人体中心跨越 `4.431/2.675/0.168 m`；两者二维包围框分别占房间地面包围框的 `47.44%/49.39%`。完整80行坐标见 `outputs/data/long_trajectory.csv`。

## 4. 并行 D0--D5 几何条件

几何种子为 `20260730..20260734`。对每个种子和时刻生成共享的独立标准正态量 `z_d,z_a,z_e`，从而使不同条件形成配对设计：

| 条件 | 距离输入 | 方位输入 | 俯仰输入 | 要隔离的因素 |
|---|---|---|---|---|
| D0 | `d_t` | `a_t` | `e_t` | 精确三维几何基准 |
| D1 | `d_t+0.02z_d` m | `a_t` | `e_t` | 小径向误差 |
| D2 | `d_t+0.10z_d` m | `a_t` | `e_t` | 大径向误差 |
| D3 | `d_t` | `a_t+6z_a`° | `e_t+3z_e`° | 仅方向误差 |
| D4 | `d_t+0.04z_d` m | `a_t+6z_a`° | `e_t+3z_e`° | 现实联合噪声档位 |
| D5 | D4基础上增加异常突发 | D4基础上增加异常突发 | D4基础上增加异常突发 | 离群/突发鲁棒性 |

D5在每个空间区块的局部索引10和11连续两帧增加：距离 `±0.18 m`、方位 `±30°`、俯仰 `±10°`；符号由几何种子和区块奇偶交替。因此每区块异常率为 `2/16=12.5%`，且异常位置在所有条件比较中固定。

所有距离裁剪到 `[0.18,0.75] m`，并统一施加两个与CSI结果无关的物理约束：手机不能位于人体盒体水平截面内；人体盒体必须完全留在房间内部。最终实际误差而非名义标准差见 `outputs/data/direct_sensitivity_summary.csv`。

## 5. 路径基与 `theta_ref`

记 `f` 为子载波频率，`b=0,1,2,3` 为反射次数，`H_b(f;r)` 为几何 `r` 下第b阶路径的复CFR之和。有效反射参数模型为：

```text
H(f;theta_ref,r) = H_0(f;r)
                 + theta_ref H_1(f;r)
                 + theta_ref^2 H_2(f;r)
                 + theta_ref^3 H_3(f;r)
```

`theta_ref` 是围绕WiTwin基准材料的每次反射共享增益修正量，不是墙体介电常数。合成真值固定 `theta_ref*=1`，校准范围为 `[0.5,1.5]`。每次校准用70次有界黄金分割搜索最小化训练区块上的复CSI均方误差。

## 6. 两层分析

### 6.1 固定参数的直接敏感性

首先固定 `theta_ref=1`，不加CSI噪声、不做参数校准，直接比较：

```text
H_true(t,f) = H(theta_ref=1,r_t)
H_Dk(t,f)   = H(theta_ref=1,r_Dk,t)
```

这样得到的直接复NRMSE、幅度RMS、相位RMS和路径计数变化只反映几何输入差异。

### 6.2 五折空间留一交叉验证

每一折完整留出Z0--Z4中的一个空间区块测试，其余4个区块共64个时刻用于校准 `theta_ref`；测试区块16个时刻完全不参与参数估计。

每折生成100次30 dB SNR复高斯CSI噪声，种子为 `20260810+fold`。100次重复按编号对5个几何种子循环分配，因此每个“测试区块×几何种子”cluster包含20次CSI重复。每个条件总计：

```text
5 folds × 100 CSI repeats = 500 results
5 zones × 5 geometry seeds = 25 paired clusters
```

D0--D5在同一折使用完全相同的带噪合成观测，测试误差与无噪声真值比较。

## 7. 指标与统计

测试复NRMSE定义为：

```text
NRMSE = sqrt(sum |H_pred-H_true|^2 / sum |H_true|^2)
```

同时报告：幅度残差RMS、包裹相位残差RMS、`|theta_hat-1|`、三维几何误差 `E_r=mean||r_est-r_true||_2`、距离误差 `E_d`、路径计数改变率。

空间结果使用两种互补汇总：

- 区块等权宏平均：先在每个测试区块×几何种子cluster内平均，再对25个cluster等权，强调最困难区域；
- 能量合并NRMSE：先合并各cluster的平方误差和真值能量再开方，避免极弱信号区因分母很小而单独支配总体数值。

两种口径都对25个配对cluster进行20,000次有放回bootstrap，种子为 `20260820`。报告同时保留逐区块结果，不把相关的CSI重复错误当作500个独立几何样本。

D1--D5的合并结果使用透明的等权条件宏平均；由于没有真实传感器发生概率，本实验不构造未经标定的“现实概率权重”。

## 8. 求解量与可复现性

主体几何求解数为：

```text
80个D0真值场景 + 5个几何种子 × 5个误差条件 × 80时刻 = 2080
```

另做5个采样收敛求解和1个重复确定性求解。最近一次完整强制运行耗时209.217秒，其中路径求解和审计179.260秒。

完整运行命令：

```bash
cd /opt/witwin
export DRJIT_LIBOPTIX_PATH=/usr/lib/x86_64-linux-gnu/libnvoptix.so.1
/opt/witwin/venv/bin/python experiments/wgeo_stage1_long_trajectory/run_experiment.py --force-sim
```

复用缓存命令见 [README.md](README.md)。

## 9. 适用边界

这是单房间、单AP、单人体盒体的内部仿真验证。误差档位是预先固定的模拟分布，不代表某款真实深度设备的实测标定；轨迹只有一条确定性路径，5个几何种子也不能代表所有空间环境。模型包含LOS和最多三次镜面反射，不包含绕射、漫散射或更高阶路径。

本实验实际使用DrJit反射后端并通过本实验审计，但这不改变官方验证器对当前固定Channel/RayD组合的 `max_bounces>0` native reflected EPC未验证限制。
