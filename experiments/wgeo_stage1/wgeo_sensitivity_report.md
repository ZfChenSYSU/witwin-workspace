# 阶段 1：`w_geo`（动态三维手机--人体相对几何误差）仿真科学假设验证报告

## 结论先行

本次阶段 1 判定为 **GO**，但结论有明确边界：在固定房间、人体盒体、WiTwin 版本、最多三次镜面反射和 2.4 GHz/20 MHz 配置下，动态三维手机--人体几何是复 CSI（channel state information，本文具体指 64 个子载波上的复数信道频率响应样本）的强影响量；把人体相对手机的位置固定为 `r_0=(0,0.40,0) m`（用房间世界坐标表示、从手机天线参考点指向人体盒体中心的名义相对向量）会造成巨大的测试误差，并把 `theta_ref`（在 WiTwin 基准材料系数之外施加的、每次反射共享的有效增益修正参数）推到搜索下界。只有真值距离而没有方向的标量模型同样失败。

最关键的数字是：

- 忽略动态几何的 SIM-B（动态观测、固定 `r_0` 预测）测试复 NRMSE（normalized root mean square error，复数归一化均方根误差）为 **0.886097**；三维 oracle SIM-C（动态观测、真值三维几何预测）为 **0.000613**。配对改善为 **0.885484**，95% bootstrap 区间 `[0.885417, 0.885548]`，相对降低 **99.93%**。
- 只有真值距离的 SIM-G（动态观测、真值距离但固定方向预测）NRMSE 为 **0.835232**；相对三维 SIM-C 的配对差为 **0.834619**，95% 区间 `[0.834553, 0.834684]`。其中 `t` 是离散时刻索引，`r_t` 是时刻 `t` 用房间世界坐标表示、从手机天线参考点指向人体盒体中心的三维向量，`d_t=||r_t||_2` 是该向量的欧氏长度，即手机--人体标量距离。因此 `d_t` 几乎无误并不意味着 `r_t` 充分。
- 含噪三维前置深度 SIM-D 的 NRMSE 为 **0.054018**，比 SIM-B 降低 **93.90%**，对应改善区间 `[0.827322, 0.836833]`。
- 常速度 RTS（Rauch--Tung--Striebel 离线平滑器）和其协方差边缘化仍优于固定几何，但在 D5（时刻 16--19 的突然几何跳变测试段）明显过度平滑：SIM-E/F 分别为 **0.767857/0.726841**。这是负结果，不应被写成“滤波必然有益”。

## 1. 详细实验设计思路

### 1.1 先明确实验真正要回答的问题

阶段 1 不是要证明“任何人体因素都会影响无线信道”，因为这一命题过宽，也无法指导论文中的方法设计。它只回答一个更窄、可证伪的问题：

> 在房间、收发机、人体体型和材料都固定时，手机与人体之间随时间变化的三维相对几何 `r_t` 是否会显著改变复 CSI；如果优化模型忽略该动态几何，是否会产生不能由静态环境参数解释的信道误差和参数偏差？

这里把总人体误差拆成两类：

- `w_geo`：距离、方位、俯仰和遮挡关系等动态几何错误，是本实验主动改变的自变量；
- `w_human`：除动态相对几何以外的人体建模误差，包括体型、衣物、介电属性和人体代理精细程度等；本实验将这些因素全部固定，不把它们误称为几何噪声。

因此，本阶段能支持的是“***是否值得把动态三维几何作为论文主线***”，不能支持“已经解决所有人体建模误差”。

### 1.2 因果隔离思路

实验遵循单一干预原则：生成观测和生成模型预测时，房间、接收机、手机轨迹、人体尺寸、人体材料、载频、带宽和求解器配置完全相同，只替换模型看到的 `r_t`。记 `t=0,...,19` 为离散时刻索引；`q_t=(p_t,R_t)` 为手机在时刻 `t` 的六自由度位姿，其中 `p_t` 是世界坐标中的三维位置，`R_t` 是由 yaw、pitch、roll 指定的姿态；`theta_ref*=1` 为合成数据采用的有效逐反射增益真值；`beta` 为本实验固定的人体盒体尺寸和材料属性集合；Tx 是手机发射端，Rx 是固定 CSI 接收端。简化后的推理链如下：

```text
固定项：房间、theta_ref*=1、人体 beta、Rx、手机 q_t、WiTwin 配置
                              │
主动变化：真实三维几何 r_t ──┤
                              ▼
             WiTwin 0--3 阶路径 ──> 无噪声复 CSI 真值
                                          │
                              加入受控 CSI 测量噪声
                                          ▼
                             各组看到不同的几何信息
                                          │
                     在训练段估计同一个 theta_ref
                                          ▼
                  在未见的 D5 测试轨迹上比较 CSI 和参数误差
```

这样设计的理由是：如果同时改变人体材质、体型或房间，SIM-B 与 SIM-C 的差异就不能唯一归因于 `w_geo`。当前设计虽然牺牲了一部分现实复杂度，却获得了清楚的因果解释。

### 1.3 为什么采用这个最小场景

场景只包含一部发射手机和一个 CSI 网卡接收端，对应后续真实系统中的 `phone -> CSI acquisition NIC` 链路。房间使用四面墙、地面和天花板，***是能够产生 LOS（line of sight，视距直达）、遮挡和多阶反射的最小封闭环境；人体使用固定长方体，是为了让人体位置可精确控制，同时避免第一轮就引入人体网格形状误差。***

手机轨迹 `q_t` 同时包含位置和姿态，并使用 dipole（偶极子）天线方向图。这样 `r_t` 的变化不只是改变路径长度，还会改变：

- 人体是否进入 Tx--Rx 的 LOS 路径；
- 各墙面和人体表面的镜面反射是否可见；
- 每条路径的反射顶点、总长度、离开角和到达角；
- 一至三阶路径在 64 个子载波上的相干叠加。

原计划要求先从 LOS 和一次反射开始。本次在预检确认 DrJit 反射后端可运行后，将上限提高到三次反射，以免只在过度简化的单反射条件下得出结论。`max_num_paths=192` 是单个场景允许保留的最大传播路径数，用作防止静默截断的安全上限；最终最大只用到 66 条。

### 1.4 为什么要设计 D0--D5，而不是只随机移动人体

纯随机轨迹虽然容易生成，却很难解释“什么类型的几何变化导致了结果”。D0--D5 是按扰动强度和动态形态逐级构造的压力测试：

| 扰动           | 设计意图               | 能检验的现象               |
| -------------- | ---------------------- | -------------------------- |
| D0：固定 40 cm | 建立数值与噪声地板     | 模型正确时能否恢复真值     |
| D1：±2 cm     | 模拟轻微手抖           | 小误差是否已能改变相位     |
| D2：±10 cm    | 模拟自然手持变化       | 常见尺度下是否需要动态几何 |
| D3：±20 cm    | 覆盖近 LOS、侧方和背向 | 遮挡与路径集合切换         |
| D4：慢周期     | 模拟手臂摆动           | 平滑动态是否适合时序处理   |
| D5：突然跳变   | 模拟换手或姿态调整     | 滤波滞后和测试泛化         |

距离、方位角和俯仰角不是同步按一个比例变化，而是有意组合成侧方、斜向、近 LOS 和背向状态。这使实验可以区分“距离变了”和“方向变了”。否则 SIM-G 的标量/三维消融没有意义。

时刻 0--15 的 D0--D4 用于训练 `theta_ref`，时刻 16--19 的 D5 完全留作测试。把最难的突变留出，是为了防止得到“模型只在参与参数估计的轨迹上拟合得好”的循环结论。当前测试集只有 4 个时刻，但每个时刻包含 64 个复子载波，并在 5 个深度种子和 200 次 CSI 重复下评估；后续仍需用更多独立轨迹扩大外部效度。

### 1.5 SIM-A--SIM-G 是一组逐层反事实对照

七组不是七个互不相关的模型，而是一条逐层排除替代解释的证据链：

| 关键比较       | 只改变什么                          | 回答的问题                                                                |
| -------------- | ----------------------------------- | ------------------------------------------------------------------------- |
| SIM-A 自身     | 什么都不失配                        | 数值实现和 30 dB SNR（signal-to-noise ratio，信噪比）下的误差地板是多少？ |
| SIM-B 对 SIM-C | 模型使用固定 `r_0` 或真值 `r_t` | 动态几何本身是否重要？                                                    |
| SIM-A 对 SIM-C | 静态或动态，但模型都知道真值几何    | 动态场景是否本身不可求解，还是只有几何失配才出错？                        |
| SIM-D 对 SIM-B | 增加带噪三维前置深度                | 现实精度的深度信息是否仍有实际价值？                                      |
| SIM-E 对 SIM-D | 增加常速度 RTS 时序先验             | 平滑是否真的优于原始深度？                                                |
| SIM-F 对 SIM-E | 从单点几何改为协方差 sigma 点平均   | 显式传播几何不确定性是否有增益？                                          |
| SIM-G 对 SIM-C | 保留真值距离但删除真实方向          | `d_t` 能否替代三维 `r_t`？                                            |

各组观测端的动态真值保持一致。例如 SIM-B 与 SIM-C 接收的是同一批动态 CSI，差异只在模型端是否使用真值 `r_t`。这使 B−C 成为本阶段最直接的 `w_geo` 因果效应估计。

SIM-E/F 没有被预设为必须优于 SIM-D。它们是可证伪的算法假设：如果运动先验错误，平滑完全可能更差。本次 D5 的结果确实否定了“普通 RTS 必然有益”，这正是设置独立组而不是只展示最佳方法的意义。

### 1.6 为什么引入 `theta_ref`，以及它怎样检测几何混杂

只比较 CSI 误差可以说明几何影响预测，但论文还关心几何错误会不会污染待估的静态环境参数。因此第一轮只校准前文定义的单一有效参数 `theta_ref`，以便进行严格的梯度和可辨识性检查。

对任意模型几何向量 `r`，记 `f` 为子载波频率，`b=0,1,2,3` 为路径反射次数，`H_b(f;r)` 为 WiTwin 在几何 `r` 下求得并汇总的第 `b` 阶复数 CFR（channel frequency response，信道频率响应）路径基；`H(f;theta_ref,r)` 是四个阶次缩放并相加后的总复数 CFR。模型写成：

```text
H(f; theta_ref, r) = H_0(f;r)
                    + theta_ref H_1(f;r)
                    + theta_ref^2 H_2(f;r)
                    + theta_ref^3 H_3(f;r)
```

合成观测采用真值 `theta_ref*=1`；由训练数据校准得到的估计值记作 `theta_hat`。一条三阶路径经历三次反射，所以受到 `theta_ref^3` 缩放。这种写法保留了“每次反射共享一个有效校正量”的物理含义，同时只增加一个自由度，便于判断可辨识性。

实验逻辑是：

1. 如果几何正确，训练数据应该在 `theta_ref=1` 附近取得唯一最小损失，即 `theta_hat` 应接近 1；
2. 如果几何错误，但 `theta_hat` 显著偏离 1，说明静态参数正在吸收几何误差；
3. 如果使用 `theta_hat` 后测试 CSI 仍然很差，说明几何失配不是一个全局增益能够替代的。

`theta_ref` 是围绕 WiTwin 基准材料的有效反射校正量，不是某面墙的介电常数。本阶段选择它是为了干净地验证“参数污染”机制，而不是提前声称完成真实材料反演。

### 1.7 为什么同时加入深度噪声和 CSI 噪声

系统未来有两条不同的误差来源：前置深度给出的几何不准确，CSI 网卡的复数测量也有噪声。如果只重复 CSI 噪声，结论会条件化在单个偶然的深度误差上；如果只重复深度噪声，又无法评估参数估计的测量稳定性。

因此本实验使用两层重复：

- 5 个独立深度噪声种子，距离、方位和俯仰标准差分别为 4 cm、6°和 3°；每个种子都重新进行真实 WiTwin 三阶路径求解；
- 200 次 30 dB 复高斯 CSI 重复，均匀分配为每个深度种子 40 次。

SIM-E 的状态是 `[r_t,v_t]`，其中 `v_t` 是按一个离散时间索引步长定义的人体相对位置变化速度；该组采用三轴常速度 Kalman 前向滤波和 RTS 后向平滑。过程噪声在运行前固定，没有查看 D5 测试结果后调参，所以它代表一个诚实但可能失配的通用时序 baseline。记 `P_t` 为 RTS 在时刻 `t` 输出的三维位置后验协方差矩阵。SIM-F 从 `P_t` 构造 1 个后验均值点和 3 对协方差主轴正负点，共 7 个等权 sigma 点；每个点都单独运行 WiTwin，再在复 CFR 层平均。它检验的是“显式传播局部几何不确定性”，不是简单平滑最终误差。

含噪、滤波和 sigma 点向量还使用同一个物理可行域约束：手机必须在水平人体盒体外至少 2 cm。这个约束在求解前统一施加，目的是阻止滤波平均把手机放入人体代理内部，而不是根据 CSI 结果选择有利位置。

比较时各方法使用同一重复编号的观测噪声和深度种子，形成配对设计。这样组间差值不被无关的噪声抽样差异放大。10,000 次 bootstrap 也对“同一重复中的组间差值”重采样，而不是把两组当成互不相关样本。

### 1.8 为什么用这些指标

- 记 `H_pred(t,f)` 和 `H_true(t,f)` 分别为预测与无噪声真值复 CFR，`T_test={16,17,18,19}` 为测试时刻集合，`F` 为 64 个子载波频率的集合。测试复 NRMSE `E_s` 是主指标，其定义为

  ```text
  E_s = sqrt(
          sum over t in T_test, f in F of |H_pred(t,f)-H_true(t,f)|^2
          / sum over t in T_test, f in F of |H_true(t,f)|^2
        ).
  ```

  它同时保留幅度和相位，并相对真值能量归一化，便于不同组比较。
- 幅度残差定义为 `20 log10(|H_pred|/|H_true|)`，包裹相位残差定义为 `angle(H_pred*conj(H_true))`；幅度 RMS 和相位 RMS 分别是这些残差在全部测试时刻和子载波上的均方根，其中相位 RMS 最终换算为度。二者用于解释 `E_s` 来源，防止只用一个复数总分掩盖相位错误。
- 参数绝对误差 `E_theta=|theta_hat-theta_ref*|=|theta_hat-1|`，用于检查静态有效参数是否被几何误差污染。
- 对任一待评估时刻集合 `T`，记 `r_est(t)` 为某方法给出的人体相对位置估计。三维向量平均误差定义为 `E_r=mean_(t in T)||r_est(t)-r_t||_2`；标量距离平均绝对误差定义为 `E_d=mean_(t in T)| ||r_est(t)||_2-d_t |`。SIM-G 正是通过 `E_d≈0`、`E_r` 很大来证明方向信息不可省略。
- `theta` 区间覆盖率是 200 次重复中名义 95% 区间 `[theta_hat-1.96 SE, theta_hat+1.96 SE]` 包含真值 1 的比例，其中 `SE` 是基于训练损失局部导数和已知 CSI 噪声方差得到的标准误；下界命中率是 `theta_hat` 落到搜索下界 0.5 的重复比例，用来识别“看似方差很小、实为估计被边界截住”的假稳定。
- 主 NRMSE 相对无噪声测试真值计算，使它测量模型泛化误差，而不是把不可预测的测试测量噪声也算作模型失败。

不同初始化没有作为主要随机因素，是因为 `theta_ref` 只有一维，并使用固定边界内的黄金分割搜索；无噪声 2001 点损失剖面进一步检查了全局极小值和局部极小值数量。若下一阶段同时估计多个材料参数，就必须重新加入多初始化和参数相关性实验。

原计划还提出检查“真实 `r_t` 是否落在 `P_t` 的预测协方差范围内”。本次虽然用 `P_t` 生成 SIM-F sigma 点，但没有把三维协方差覆盖率作为 Go 指标：5 个深度种子只产生 100 个几何状态，且常速度模型在 D5 已明显失配，此时直接报告名义椭球覆盖率容易给出虚假的精确印象。这是一项尚未完成的校准检查，下一轮应增加深度种子和轨迹数量后，用三维 Mahalanobis 距离或 NIS（normalized innovation squared，归一化新息平方）覆盖率单独报告。

### 1.9 从数据到结论的判定顺序

为了避免看到结果后选择有利指标，结论按以下顺序作出：

1. 先检查三阶路径是否真的出现、路径是否触及上限、重复求解是否一致、逐阶路径基是否能重建 WiTwin CFR；数值审计不通过就不解释科学结果。
2. 再检查 SIM-A/C 是否接近噪声地板、`theta_ref` 是否恢复到 1；这一步验证实验实现和参数可辨识性。
3. 用 B−C 判断忽略动态几何的主效应，并要求改善出现在未参与拟合的 D5 测试段。
4. 用 G−C 判断标量距离是否足够；如果差异不显著，就没有理由坚持三维前置深度。
5. 用 D−B 判断带噪深度是否有工程价值，再用 E−D、F−E 分别检验时序先验和不确定性传播。
6. 最后才应用预先写定的 Go/No-Go 门槛。负结果会保留，不因总体 GO 而删除。

以某个动态时刻为例：观测 CSI 始终由真实 `r_t` 的 WiTwin 路径生成；SIM-B 在同一手机 `q_t` 下把人体错误地放到固定 `r_0`，SIM-C 放到真值 `r_t`，SIM-G 只把人体放到正确半径但错误方向。三组都只能用训练段调整同一个 `theta_ref`，然后在 D5 测试段比较预测。这就是 B−C 和 G−C 能分别归因于“动态几何缺失”和“方向缺失”的原因。

### 1.10 设计边界

这是一项受控的内部效度实验。它优先保证变量可解释，不追求一次覆盖所有现实因素。单一房间、单一接收机、固定人体盒体、无绕射/漫散射和最多三次镜面反射限制了外部效度；5 个深度种子也不能代表所有传感器分布。当前 bootstrap 以 200 个配对重复为重采样单位，其中每 40 次共享一个深度种子，所以区间主要表示该五种子设计内的条件性不确定度，不是任意深度传感器分布的总体置信区间。后续扩大深度种子时应使用按种子分层或聚类的 bootstrap。阶段 1 的正确用法是决定是否继续投入动态三维几何主线，并筛出下一阶段必须改进的环节，而不是把仿真数字直接当作真实硬件性能。

完整数值常量、轨迹坐标、材料和随机种子见 [simulation_config.md](simulation_config.md)。

## 3. 配对效应证据

对每个 CSI 噪声重复，先定义配对差值 `Delta=参考组的 E_s-候选组的 E_s`，所以正值表示候选组误差更低。表中的配对效应量 `d_z=mean(Delta)/std(Delta)`，其中标准差按 200 个配对差值计算；95% bootstrap 区间则来自对这些配对差值进行 10,000 次有放回重采样后的均值分布。

| 参考组减候选组 | NRMSE 平均改善 |   95% bootstrap 区间 | 相对改善 |       配对效应 `d_z` |
| -------------- | -------------: | -------------------: | -------: | ---------------------: |
| SIM-B − SIM-C |       0.885484 | [0.885417, 0.885548] |   99.93% |                1896.04 |
| SIM-B − SIM-D |       0.832079 | [0.827322, 0.836833] |   93.90% |                  23.87 |
| SIM-B − SIM-E |       0.118240 | [0.103037, 0.133183] |   13.34% |                   1.08 |
| SIM-B − SIM-F |       0.159256 | [0.149465, 0.168692] |   17.97% |                   2.29 |
| SIM-B − SIM-G |       0.050865 | [0.050865, 0.050865] |    5.74% | 边界删失，不解释其大小 |

所有关键 Go 比较的区间下界均大于 0。极大的 `d_z` 出现在 oracle 或参数边界导致组内方差极小的比较中，不能被解释为跨场景总体效应；更稳妥的证据是绝对差值、相对改善和 bootstrap 区间。

## 8. 必须保留的负结果和限制

1. 常速度 RTS 对 D5 距离/方向突变不匹配，测试 `E_r` 从原始深度的 0.05646 m 恶化到 0.28295 m。SIM-E 仍比完全固定 `r_0` 好，但远差于直接使用含噪三维深度。
2. SIM-F 只比 SIM-E 改善 5.34%。围绕一个有偏均值做协方差边缘化不能修复运动模型偏差；下一阶段应试验 jump-aware、IMM 或鲁棒状态模型。
3. 本阶段的 `theta_ref` 是有效逐反射增益，不是墙体介电常数。这里证明的是“几何误差会污染静态有效参数”，还没有证明真实材料参数的无偏恢复。
4. 统计覆盖 5 个深度种子和 200 次 CSI 重复，但场景、人体代理和接收机位置仍是单一受控族；不能把区间外推到任意房间或真实设备。
5. 求解包含 LOS 和最多三次镜面反射，绕射、漫散射和更高阶路径不在本次因果隔离范围内。
6. 官方验证器中的 native reflected EPC 限制仍然保留；本报告只验证这里实际使用的 DrJit 反射执行路径。
