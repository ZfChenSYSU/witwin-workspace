# `theta_ref` 静态参数偏差报告

## 1. 参数是什么

本阶段只选择一个容易解释且能够严格做梯度检查的静态参数：有效逐反射增益 `theta_ref`。WiTwin 先在固定材料配置下生成 0--3 阶复路径基，然后使用

```text
H(theta_ref, r) = H_0(r) + theta_ref H_1(r)
                + theta_ref^2 H_2(r) + theta_ref^3 H_3(r)
```

做校准，真值 `theta_ref*=1.0`，估计边界 `[0.5,1.5]`。因此它表示对基准反射系数的有效校正，不是某面墙的 `eps_r`，也不应被称作真实材料介电常数。

## 2. 梯度检查

在动态真值训练集、`theta_ref=0.91` 和中心差分步长 `1e-5` 下：

| 项目 | 数值 |
|---|---:|
| PyTorch 自动微分 | `-1.3774265804361912e-7` |
| 中心有限差分 | `-1.377426579466075e-7` |
| 相对误差 | `7.04296e-10` |
| 通过阈值 | `< 1e-4` |
| 判定 | 通过 |

自动微分发生在 WiTwin 逐阶路径基之上的 `theta_ref^b` 校准层；本报告没有宣称人体位置本身已经通过 WiTwin 端到端自动微分。

## 3. 可辨识性

对 `[0.5,1.5]` 使用 2001 点无噪声损失剖面：

- 全局最小值位置为 `theta_ref=1.000`；
- 内部局部极小值只有 1 个；
- 真值附近二阶曲率为 `1.8389899979e-6 > 0`。

![参数损失剖面](outputs/figures/theta_loss_profiles.png)

这证明在 oracle 几何和当前训练激励下，该有效参数不是平坦方向。它不证明其他材料参数与几何同时自由变化时仍可辨识。

## 4. 几何误差怎样污染参数

| 组 | 几何模型 | `theta_hat` 均值±标准差 | 平均绝对偏差 | 95%覆盖率 | 下界命中率 |
|---|---|---:|---:|---:|---:|
| SIM-A | 静态真值 | 1.000423 ± 0.003481 | 0.002799 | 0.930 | 0.000 |
| SIM-B | 动态观测、固定 `r_0` | 0.500000 ± 0.000000 | 0.500000 | 0.000 | 1.000 |
| SIM-C | 动态 oracle | 1.000122 ± 0.002942 | 0.002339 | 0.965 | 0.000 |
| SIM-D | 含噪三维深度 | 0.961465 ± 0.040932 | 0.038535 | 0.195 | 0.000 |
| SIM-E | 常速度 RTS | 0.767572 ± 0.329194 | 0.367572 | 0.000 | 0.600 |
| SIM-F | 协方差边缘化 | 0.500000 ± 0.000000 | 0.500000 | 0.000 | 1.000 |
| SIM-G | 只有真值距离 | 0.500000 ± 0.000000 | 0.500000 | 0.000 | 1.000 |

SIM-A/C 的估计围绕 1，覆盖率接近名义 95%；说明实现、噪声标定和 Fisher 区间在模型正确时相互一致。SIM-B/G 则 100% 命中 0.5 下界，表明优化器试图削弱全部反射来补偿错误路径几何。SIM-E/F 的有偏中心也造成同类吸收。

SIM-D 没有触界，但平均偏差 0.038535，且覆盖率仅 19.5%。即使 CSI 测试 NRMSE 已明显下降，厘米级/角度级深度误差仍会让静态参数的名义置信区间失真；因此“预测够好”和“物理参数无偏”必须分别检验。

## 5. 对科研假设的结论

支持以下受限结论：

1. 当 `w_geo` 正确时，有效静态反射参数可以稳定恢复；
2. 当 `w_geo` 错误时，参数估计会系统性吸收几何误差，甚至被推到边界；
3. 参数区间覆盖率能暴露模型失配，但边界命中时普通 Fisher 区间已经不再可信。

下一阶段若改估真实 `eps_r/sigma_e`，必须重新进行端到端梯度检查、联合损失剖面、参数相关矩阵和多初始化恢复，不能沿用本报告的单参数可辨识性结论。
