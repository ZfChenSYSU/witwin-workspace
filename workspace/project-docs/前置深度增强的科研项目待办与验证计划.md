## 1. 可能的 Main Contributions

以下贡献均是“待实验建立的候选贡献”，不是当前已经获得的结论。论文最终能否采用，取决于对应的验证条件是否通过。

### Contribution 1：揭示动态人体—设备几何对无线环境反演的系统性影响

系统研究移动手持采集过程中人体—手机相对几何：

```math
\mathbf r_t = \mathbf p^{A}_{B,t}
```

及其距离：

```math
d_t = \|\mathbf r_t\|_2
```

对 CSI 幅度、相位、路径结构和环境参数估计 `\theta` 的影响，重点证明：

> 当算法错误地把人体固定在手机后方某个默认距离 `d_0` 时，动态人体误差是否会被环境参数吸收，从而使墙体介电参数、反射系数等静态环境参数产生系统性偏差。

这里需要严格区分：

- `d_t` 或 `\mathbf r_t`：真实人体—手机几何状态；
- `w_geo`（原计划中的 `w_hd`）：人体几何建模错误在 CSI 中诱发的误差；
- `\theta`：需要反演的静态环境参数。

该贡献成立需要同时满足：

- 仿真中改变 `\mathbf r_t` 会造成稳定、可解释的 CSI 变化；
- 错误固定 `d_0` 会显著增大 `E_\theta`；
- 真实受控实验中不同人体—手机几何会引起可重复的 CSI 差异；
- 改用测得的动态几何后，测试轨迹而非仅训练轨迹的结果得到改善。

### Contribution 2：前置深度条件下的不确定性感知可微射线追踪方法

提出一种由手机前置深度/人脸三维跟踪提供动态人体几何观测的无线环境反演方法。该方法不把 `d_t` 当作完全依赖 CSI 反推的隐变量，而将其建模为带协方差的外部传感器观测：

```math
\mathbf z_t^{depth} = \mathbf r_t + \mathbf v_t,
\qquad \mathbf v_t \sim \mathcal N(0,R_t)
```

算法包括：

- 前摄像头—Wi-Fi 天线外参标定；
- 人脸—胸腔参考点的个体或群体标定；
- 异常检测和质量感知的三维时序滤波；
- 将 `\hat{\mathbf r}_t` 作为射线追踪条件输入；
- 根据 `P_t` 对人体几何进行采样边缘化或不确定度加权；
- 可选的视觉观测约束下 `\mathbf r_t` 与 `\theta` 联合优化。

真正可能形成方法创新的部分是：

> 把动态人体几何观测及其不确定度显式传播到无线可微射线追踪中，从而分离或缓解人体动态几何诱发的模型失配，减少其对静态环境参数估计的污染。

前置深度主要处理人体几何误差 `w_geo`（原计划中的 `w_hd`），不能直接消除由体型、衣物、介电属性和未建模姿态引起的 `w_human`（原计划中的 `w_hh`）。相关贡献和实验结论必须限定在“人体动态几何误差”范围内。

普通卡尔曼滤波、RTS 平滑、Huber 损失或双摄采集本身不是主要创新点，它们是构成上述方法的工具。

该贡献成立需要满足：

- 前置深度产生的 `\mathbf r_t` 达到可接受的静态和动态精度；
- DEPTH-KF/DEPTH-RTS 优于固定 `d_0`；
- 使用协方差的边缘化或软约束优于把深度结果当无误差真值；
- `\theta` 的跨轨迹、跨轮次一致性得到改善。

### Contribution 3：低成本、链路语义严格的手机多模态无线数字孪生采集框架

构建一套同步采集和对齐以下数据的低成本框架：

- 后摄像头/IMU/SLAM 手机位姿；
- 前置 TrueDepth 或人脸三维锚点；
- CSI 采集网卡接收或监听手机通信帧时输出的 Wi-Fi CSI；
- 人体几何标定真值；
- 可供射线追踪使用的真实房间网格和材料区域；
- 房间、CSI 采集网卡和手机天线的统一坐标。

框架必须明确 CSI 采集网卡测量的是手机发出的通信帧在“手机到采集网卡”这一物理链路上的 CSI：

```text
phone -> CSI acquisition NIC
```

而不是误称为：

```text
phone -> router/AP
```

该贡献的价值不只是“收集多种传感器数据”，而是建立一个物理链路定义正确、时间坐标统一、人体动态几何可观测、能够进入可微射线追踪闭环的数据管线。

这里的“与手机通信的 CSI”是指采集网卡针对其接收或监听到的手机通信帧进行 CSI 测量；普通路由器可以承担流量目标或网络接入功能，但采集网卡输出的 CSI 不能表述为普通路由器端 CSI。

该贡献成立需要满足：

- CSI、SLAM 和人体几何可以在统一时间轴上对齐；
- 主线数据只包含物理含义一致的手机上行 CSI；
- 完成多轮、多人或多轨迹的可重复采集；
- 数据格式、标定方法和质量指标完整记录。
- 给出设备清单、总成本和与 USRP/SDR 方案的成本及部署复杂度对照，以量化“低成本”而非只作定性描述。

### 支撑性验证体系：动态人体几何误差基准与消融

通过固定距离、自然手持、错误几何、时间错位、跨人替换等实验，建立动态人体几何对无线数字孪生影响的系统消融基准。

推荐包含以下关键消融：

- 固定 `d_0` 与真实动态 `\mathbf r_t`；
- 只使用标量 `d_t` 与使用三维向量 `\mathbf r_t`；
- 原始前摄观测、在线 KF、离线 RTS；
- 忽略协方差与不确定度边缘化；
- 正确 `d_t`、时间打乱的 `d_t`、其他人的 `d_t`；
- 仅幅度损失与幅度—相位分离损失；
- 普通 MSE 与鲁棒损失。

本计划默认把它作为 Contribution 1–3 的验证体系，而不单列为 Main Contribution。只有当最终数据覆盖多人、多轨迹、多衣物和多轮重复采集，且具有完整格式、基准实现并计划公开时，才重新评估是否将其提升为数据集或 benchmark 贡献。

### 不建议单独声称为 Main Contribution 的内容

- 使用成熟 SLAM 获取手机轨迹；
- 同时开启手机前后摄像头；
- 使用 TrueDepth、ARKit、AX210 或 PicoScenes；
- 将普通 KF、EKF、UKF 或 RTS 直接用于 `d_t`；
- 单独使用 Huber/Cauchy 损失；
- 仅搭建 WiTwin 或其他射线追踪环境；
- 只证明人体存在会改变 CSI。

这些工作可以成为系统组件、baseline 或工程贡献，但必须与动态几何误差、环境参数偏差和不确定度传播结合，才能形成有说服力的研究贡献。

## 2. 问题背景与形式化定义

### 2.1 室内无线信道建模背景

现有室内信道建模大致可以分为四类：

1. **统计信道模型**：计算成本低，但对具体房间、具体位置和传播路径的解释能力有限；
2. **确定性射线追踪**：具有明确几何和电磁传播含义，但精度依赖房间几何、材料参数和传播模型；
3. **纯数据驱动模型**：能够从大量数据拟合复杂信道，但通常缺少物理可解释性，并依赖训练分布；
4. **物理模型与数据驱动联合方法**：利用实测无线数据校正物理模型或环境参数，在可解释性和数据适配能力之间取得折中。

本项目属于第四类：使用实测 CSI 监督可微射线追踪，使房间材料参数等静态环境参数 `\theta` 逐步适配真实室内信道。

### 2.2 现有采集方式的成本与移动性问题

已有高精度无线环境重建研究常使用 USRP/SDR、固定测点或专用同步设备。这类方案具有良好的可控性，但设备成本、部署复杂度和采集时间较高，也不容易扩展到普通用户携带设备的移动采集。

本项目希望验证一条低成本替代路线：

- 以手机作为移动无线发射端；
- 手机后摄像头和 IMU 为成熟 SLAM/VIO 提供位姿与场景重建输入；
- 手机前置深度/人脸三维跟踪提供动态人体—手机几何；
- 使用支持 CSI 的低成本网卡采集手机通信帧对应的 CSI；
- 将上述数据用于室内无线可微射线追踪的参数优化。

“低成本”必须通过设备 BOM、总成本、部署步骤和采集效率与 USRP/SDR 路线对比后才能成立，不能只根据使用了手机和网卡直接下结论。

### 2.3 修正后的采集场景与 CSI 链路

研究对象是几何和主要材料在采集期间基本不变的室内空间。实验人员手持手机移动，系统采集：

1. **CSI**：CSI 采集网卡接收或监听手机发出的通信帧，输出该帧在 `phone -> CSI acquisition NIC` 链路上的 CSI；
2. **后摄视频与 IMU**：用于成熟 SLAM/VIO 的手机六自由度位姿，并为真实房间三维模型提供输入；
3. **前置深度/人脸三维锚点**：用于估计人体相对手机天线的三维位置；
4. **外部标定数据**：用于验证前摄人体几何、时间同步和坐标对齐精度。

普通路由器/AP 可以用于建立局域网和承载手机流量，但不能把 CSI 采集网卡输出的数据称为普通路由器端 CSI。主线物理拓扑为：

```text
手机产生无线通信帧
  ├─ 普通 AP/网络端：承载通信流量（如实验需要）
  └─ CSI 采集网卡：接收或监听手机帧并输出 CSI

可微射线追踪建模链路：phone -> CSI acquisition NIC
```

各设备通常不共享同一硬件时钟，因此不能假设 CSI、视频、IMU 和前置深度天然具有“同一时间戳”。正确目标是记录各自本地时间戳，并通过同步事件、时钟偏移和漂移估计映射到统一时间轴。

### 2.4 人体引起的模型失配

采集者必须靠近并手持手机，人体会吸收、遮挡、反射或散射无线信号。若仿真只使用一个固定人体模型，实测和仿真之间会出现至少两类人体相关失配：

- **动态几何失配 `w_geo`**：人体相对手机的距离、方向和姿态随时间变化，对应原计划中的 `w_hd`；
- **人体属性失配 `w_human`**：体型、衣物、介电参数、精细姿态和简化人体几何造成的误差，对应原计划中的 `w_hh`。

前置深度主要用于观测和降低 `w_geo`。`w_human` 通常与人员、衣物和传播路径相关，不能未经验证直接并入零均值高斯噪声。主实验应先固定人员和衣物研究 `w_geo`，再通过跨人员、跨衣物实验研究剩余人体属性误差。

### 2.5 统一变量与前向模型

手机状态不应只用三维位置 `loc` 表示。定义：

```math
\mathbf q_t=(\mathbf p_t,R_t)\in SE(3)
```

其中 `\mathbf p_t` 和 `R_t` 分别为手机位置与姿态。定义人体胸腔参考点相对手机 Wi-Fi 天线的三维位置：

```math
\mathbf r_t=\mathbf p^A_{B,t},
\qquad d_t=\|\mathbf r_t\|_2
```

将未被 `\mathbf r_t` 表达的人体属性记为：

```math
\boldsymbol\beta_t=
\{\text{body shape},\text{body pose},
\text{clothing},\text{electromagnetic properties}\}
```

实测 CSI 应表示为多子载波、多天线或多空间流构成的复向量：

```math
\mathbf s_t\in\mathbb C^M
```

而不是单一复标量。可微射线追踪前向模型写为：

```math
\hat{\mathbf s}_t=
F_\theta(\mathbf q_t,\mathbf r_t,
\boldsymbol\beta_t;\mathbf c)
```

其中：

- `\theta`：待优化的静态环境参数，例如墙面材料参数；
- `\mathbf c`：载频、带宽、子载波、接收网卡位置、天线配置等固定实验条件；
- `\mathbf q_t`：手机六自由度位姿；
- `\mathbf r_t`：可由前置深度观测的动态人体几何；
- `\boldsymbol\beta_t`：不能完全观测的人体属性与姿态细节。

### 2.6 残差与误差来源

测量—仿真残差可以概念性拆分为：

```math
\mathbf e_t=
\mathbf s_t^{meas}-\hat{\mathbf s}_t
=\mathbf w_{gauss}
+\mathbf w_{geo}
+\mathbf w_{human}
+\mathbf w_{hw}
+\mathbf w_{sim}
```

其中：

- `\mathbf w_{gauss}`：随机测量噪声；
- `\mathbf w_{geo}`：人体动态几何未正确建模造成的误差；
- `\mathbf w_{human}`：人体体型、衣物、材质和精细姿态误差；
- `\mathbf w_{hw}`：增益、CFO、SFO、PDD、相位偏移等 CSI 硬件误差；
- `\mathbf w_{sim}`：房间几何、材料模型、天线模型和传播近似造成的 sim-to-real gap。

这一加法只用于实验设计和误差归因，不表示各误差严格独立或在完整非线性传播模型中可以精确相加。实验必须通过固定人员/衣物、无人参考、硬件校准和仿真真值等对照尽量隔离不同误差来源。

### 2.7 原始研究问题的更新

原始问题是如何减小实测 CSI 与射线追踪 CSI 的差距，并稳定优化环境参数。结合前置深度方案后，具体化为：

1. 动态人体几何是否对 CSI 和 `\theta` 估计产生显著、可重复的影响？
2. 前置深度能否以足够精度和有效率观测 `\mathbf r_t`？
3. 将 `\hat{\mathbf r}_t` 及其协方差传播到可微射线追踪后，能否降低测试残差和环境参数偏差？
4. 在控制动态几何后，剩余的 `w_human`、`w_hw` 和 `w_sim` 各自占多大影响？
5. 低成本手机与 CSI 网卡方案在成本、部署和精度上相对 USRP/SDR 的实际权衡是什么？

## 3. 更新后的研究定位

### 3.1 核心科学问题

本项目集中回答三个问题：

1. **显著性**：动态人体—手机相对几何是否会对 CSI 产生稳定可观测影响？
2. **偏差性**：忽略该动态几何是否会导致静态环境参数 `\theta` 被错误更新？
3. **可修正性**：使用前置深度观测及其不确定度后，能否降低 CSI 残差和 `\theta` 偏差？

### 3.2 研究对象从标量扩展为三维几何

原计划主要关注标量距离 `d_t`。新计划优先保留：

```math
\mathbf r_t=[x_t,y_t,z_t]^T
```

并将：

```math
d_t=\|\mathbf r_t\|_2
```

作为派生量。原因是相同距离、不同方向的人体位置对遮挡和人体反射路径的影响不同。

初期仿真仍可先使用标量 `d_t` 降低实现难度，但进入真实闭环前必须比较标量和三维模型。

### 3.3 `d_t` 不再优先作为 CSI 隐变量

新优先级为：

```text
前置深度直接观测人体几何
  > 前置深度观测 + 时序滤波
  > 深度协方差传播/边缘化
  > 深度约束下的联合优化
  > 仅根据 CSI 估计 d_t
```

只有在前置深度丢失或不可用时，才把 CSI 辅助的 `d_t` 估计作为扩展方向。EKF/UKF 不再是默认主线。

### 3.4 SLAM 的定位保持不变

后摄像头和 IMU 使用成熟 SLAM/VIO 系统输出手机位姿；真实房间网格使用成熟三维重建、扫描或人工 CAD 测量流程准备。项目只验证：

- SLAM 是否稳定；
- 手机轨迹能否与 CSI 和前置深度同步；
- SLAM 坐标能否与房间坐标对齐；
- 手机天线相对相机的固定外参能否标定。
- 房间模型的尺度、主要传播表面和材料区域能否满足射线追踪输入要求。

不再把重新设计 SLAM/IMU 滤波作为主线贡献。

## 4. 核心假设与判定标准

### 4.1 科学假设

- [ ] H-SCI-1：改变 `d_t` 或 `\mathbf r_t` 会产生超过测量/仿真噪声的 CSI 幅度或相位变化。
- [ ] H-SCI-2：人体几何影响随 LOS 遮挡、人体反射和空间位置发生可解释变化。
- [ ] H-SCI-3：错误固定 `d_0` 会显著增大环境参数误差 `E_\theta`。
- [ ] H-SCI-4：使用真实或测得的 `\mathbf r_t` 能显著降低测试轨迹 CSI 残差。
- [ ] H-SCI-5：使用动态人体几何能降低不同采集轮次估计出的 `\theta` 方差。
- [ ] H-SCI-6：三维 `\mathbf r_t` 比只使用标量 `d_t` 更能解释人体遮挡和反射差异。
- [ ] H-SCI-7：人体几何协方差边缘化比硬使用单点估计更稳健。
- [ ] H-SCI-8：在固定人员、衣物和主要姿态条件后，观察到的改进主要来自 `w_geo` 的降低，而不是硬件漂移或其他数据差异。
- [ ] H-SCI-9：控制 `w_geo` 后，跨人员和跨衣物实验中的剩余残差能够用于刻画 `w_human`，但不被误称为高斯噪声。

### 4.2 前置深度与算法假设

- [ ] H-DEP-1：目标手机支持后摄世界跟踪和前摄用户人脸跟踪并发。
- [ ] H-DEP-2：自然手持和行走时，人脸跟踪有效率足够高。
- [ ] H-DEP-3：人脸三维位置经标定后可以近似胸腔参考点。

## 5. 总体技术路线与优先级

```text
主线 A：w_geo 科学假设仿真
  -> d_t/r_t 扰动是否改变 CSI
  -> 是否污染 theta
  -> 决定完整人体几何建模的价值

主线 B：手机前置深度最小闭环
  -> ARKit 双向跟踪能力
  -> 人脸三维观测
  -> 人脸—胸腔/相机—天线标定
  -> KF/RTS 与不确定度输出

主线 C：CSI 硬件最小闭环
  -> AX210/AX200 + PicoScenes
  -> 手机上行帧分离
  -> 明确 phone -> CSI acquisition NIC

主线 D：多模态同步和真实闭环
  -> CSI + SLAM + r_t
  -> 固定人体 baseline
  -> 深度条件射线追踪
  -> 不确定度边缘化与联合优化

主线 E：房间模型与可微射线追踪能力
  -> 真实房间网格和材料区域
  -> 反射/遮挡路径能力
  -> theta 参数化与有限差分梯度检查
  -> 合成数据可辨识性
```

当前总体优先级：

```text
P0：w_geo 仿真显著性/偏差验证
  || 前置深度 API 与动态可用性验证
  || CSI 硬件最小可行性确认
  || 射线追踪路径/梯度/参数化能力确认

P1：人体几何标定与 KF/RTS
  || CSI 物理链路和稳定性验证

P2：三类数据时间同步、坐标对齐和真实房间模型准备
  -> 仿真中的深度观测算法对照

P3：真实 CSI + SLAM + 人体几何闭环
  -> 不确定度边缘化
  -> 可选联合优化
```

三项 P0 工作可以并行，但均应以最小可行实验为目标，避免在科学假设未成立前过度扩展系统。

## 7. 阶段 1：`w_geo` 仿真科学假设验证

这是决定论文科学主线是否成立的第一阶段。

### 7.1 最小仿真场景

- [ ] 一个发射手机和一个 CSI 采集网卡接收端。

- [ ] 一个圆柱体、椭球体或长方体人体模型。
- [ ] 固定环境参数真值 `\theta^*`。
- [ ] 第一轮只选 1–2 个可解释且通过梯度检查的环境参数。
- [ ] 固定人体体型、衣物、材质和非目标姿态变量 `\boldsymbol\beta`，先隔离 `w_geo`。
- [ ] 固定默认人体位置 `\mathbf r_0`，其距离可先设为 40 cm。
- [ ] 先完成 LOS 场景，再加入一次反射和人体遮挡/反射。

### 7.2 人体几何扰动

标量 baseline：

| 编号 | `d_t` 设置     | 目的          |
| ---- | ---------------- | ------------- |
| D0   | 固定 `d_0`     | 理想 baseline |
| D1   | `d_0 ± 2 cm`  | 轻微抖动      |
| D2   | `d_0 ± 10 cm` | 自然手持      |
| D3   | `d_0 ± 20 cm` | 极端变化      |
| D4   | 缓慢周期变化     | 手臂摆动      |
| D5   | 突然跳变         | 换手/姿态调整 |

三维增强：

- [ ] 固定距离、改变方位角；
- [ ] 固定距离、改变俯仰角；
- [ ] 同时改变距离和方向；
- [ ] 比较人体位于手机正后方、侧方和斜后方；
- [ ] 比较相同 `d_t`、不同 `\mathbf r_t` 的 CSI。

### 7.3 实验组

| 组别  | 观测数据中的人体几何 | 优化模型中的人体几何           | 目的                   |
| ----- | -------------------- | ------------------------------ | ---------------------- |
| SIM-A | 固定 `\mathbf r_0` | 固定 `\mathbf r_0`           | 理想 baseline          |
| SIM-B | 动态 `\mathbf r_t` | 固定 `\mathbf r_0`           | 测量忽略人体动态的偏差 |
| SIM-C | 动态 `\mathbf r_t` | 使用真值 `\mathbf r_t`       | 理论上限               |
| SIM-D | 动态 `\mathbf r_t` | 带噪观测 `\mathbf z_t`       | 模拟前置深度           |
| SIM-E | 动态 `\mathbf r_t` | 滤波后的 `\hat{\mathbf r}_t` | 验证时序处理           |
| SIM-F | 动态 `\mathbf r_t` | 根据 `P_t` 采样边缘化        | 验证不确定度传播       |
| SIM-G | 动态 `\mathbf r_t` | 只使用标量 `d_t`             | 与三维几何对比         |

### 7.4 评价指标

```math
E_s = \frac{1}{T}\sum_t
\|\mathbf s_t-\hat{\mathbf s}_t\|
```

```math

```

```math
E_r=\frac{1}{T}\sum_t\|\hat{\mathbf r}_t-\mathbf r_t\|_2
```


## 12. 阶段 6：深度条件的可微射线追踪算法

### 12.1 `\theta` 参数化、梯度与可辨识性

- [ ] 建立 `theta_schema.yaml`，记录每个参数对应的材料区域、物理含义、单位、上下界和先验。
- [ ] 第一轮只优化一个材料区域的一个参数；稳定恢复后再增加到 2 个参数。
- [ ] 使用合成真值 `\theta^*` 生成 CSI，并从多个初始化恢复 `\theta^*`。
- [ ] 对每个可优化参数执行中心有限差分梯度检查。
- [ ] 检查不同参数方向的 Jacobian 列相关性、奇异值或条件数。
- [ ] 检查 `\theta` 与 `\mathbf r_t`、人体材质参数之间的梯度相关性。
- [ ] 使用无人数据、参数先验或分阶段优化降低不可辨识性。
- [ ] 预先划分训练和测试轨迹，不能根据测试结果反复选择 `\theta` 参数子集。
- [ ] 若绝对相位尚未校准，先基于幅度或稳定的相对特征验证参数恢复。
- [ ] 只有低维参数恢复稳定后，才扩大 `\theta` 维度。

### 12.2 基础信道损失

实现并比较：

- [ ] 复数 MSE；
- [ ] 幅度 MSE；
- [ ] wrap-aware 相位损失；
- [ ] 幅度—相位分离损失；
- [ ] Huber/Cauchy 鲁棒损失；
- [ ] MAD/质量驱动的异常点降权。

推荐基础形式：

```math
L_{CSI}=
\lambda_A\rho_A(A_t-\hat A_t)
+\lambda_\phi\rho_\phi(
\operatorname{wrap}(\phi_t-\hat\phi_t))
```

### 12.3 人体几何算法组

| 方法        | 人体几何输入                      | 用途             |
| ----------- | --------------------------------- | ---------------- |
| FIXED       | 固定 `\mathbf r_0/d_0`          | 原始 baseline    |
| DEPTH-RAW   | 原始深度换算值                    | 测量 baseline    |
| DEPTH-KF    | 在线三维 KF                       | 主在线方法       |
| DEPTH-RTS   | 离线 RTS                          | 离线性能上限之一 |
| DEPTH-MARG  | `N(\hat{\mathbf r}_t,P_t)` 采样 | 不确定度感知方法 |
| DEPTH-JOINT | 深度约束下联合优化                | 可选增强         |
| SCALAR      | 只使用 `d_t`                    | 三维建模消融     |

### 12.4 直接条件输入

```math
\hat{\mathbf s}_t=F_\theta(
\mathbf q_t,
\hat{\mathbf r}_t)
```

- [ ] 只优化 `\theta`。
- [ ] 比较 FIXED、DEPTH-RAW、DEPTH-KF 和 DEPTH-RTS。
- [ ] 在训练轨迹和测试轨迹分别评价。

### 12.5 不确定度边缘化

```math
\mathbf r_t^{(k)}
\sim\mathcal N(
\hat{\mathbf r}_t,P_t)
```

```math
L_t\approx\frac{1}{K}\sum_{k=1}^{K}
L_{CSI}(\mathbf s_t,F_\theta(\mathbf q_t,\mathbf r_t^{(k)}))
```

- [ ] 测试不同采样数 `K`。
- [ ] 比较对角协方差和完整三维协方差。
- [ ] 检查协方差是否需要温度缩放校准。
- [ ] 比较单点输入和边缘化的计算成本。

### 12.6 深度约束联合优化（可选）

```math
L=
\sum_tL_{CSI}(\mathbf s_t,F_\theta(\mathbf q_t,\mathbf r_t))
+\lambda_v\sum_t
\|\mathbf r_t-\hat{\mathbf r}_t^{depth}\|_{P_t^{-1}}^2
+\lambda_s\sum_t\|\mathbf r_t-\mathbf r_{t-1}\|^2
+\lambda_\theta R(\theta)
```

- [ ] 先固定 `\theta`，检查 `\mathbf r_t` 是否只在视觉不确定度范围内调整。
- [ ] 再尝试交替优化 `\theta` 与 `\mathbf r_t`。
- [ ] 防止 CSI 将人体几何拉向不符合视觉观测的位置。
- [ ] 必须保留 DEPTH-KF 和 DEPTH-MARG 作为更简单对照。

### 13.2 误差归因控制

- [ ] 固定手机、采集网卡和房间，重复采集以估计硬件漂移 `w_hw`。
- [ ] 固定人员、衣物和主要姿态，只改变 `\mathbf r_t`，隔离 `w_geo`。
- [ ] 固定 `\mathbf r_t`，改变人员或衣物，评估 `w_human`。
- [ ] 固定全部几何并在不同时段重复，检查环境变化和 `w_sim`/未控因素。
- [ ] 使用支架无人数据锚定 `\theta`，避免人体参数与环境参数同时漂移。
- [ ] 每个实验条件至少重复多轮，并随机化实验顺序，避免时间漂移与方法条件绑定。
- [ ] 保存 CFO、SFO、RSSI、MCS、带宽、温度或其他可获得的硬件状态字段。

### 13.3 每组至少比较的方法

- FIXED + 普通 MSE；
- FIXED + 幅度—相位鲁棒损失；
- DEPTH-RAW；
- DEPTH-KF；
- DEPTH-RTS；
- DEPTH-MARG；
- DEPTH-JOINT（若实现）。

### 13.4 关键消融

- [ ] 正确人体几何 vs 固定 `d_0`。
- [ ] 三维 `\mathbf r_t` vs 标量 `d_t`。
- [ ] 正确时间对齐 vs 随机时间错位。
- [ ] 本人的人体轨迹 vs 其他人的轨迹。
- [ ] 正确协方差 vs 固定过小/过大协方差。
- [ ] KF vs RTS，区分实时与离线上限。
- [ ] 幅度 only vs 幅度 + 校准相位。
- [ ] 有无人数据锚定 `\theta` vs 无锚定。
- [ ] 同一人体几何的跨时段重复，量化硬件/环境漂移下限。
- [ ] 固定几何下跨人员和跨衣物，量化前置深度无法消除的 `w_human`。

### 13.5 评价标准

若环境参数有真值或仿真真值：

- `E_\theta`；
- `E_s`；
- `E_r/E_d`；
- 收敛速度和不同初始化稳定性。

若真实环境参数无精确真值：

- 训练轨迹残差；
- 测试轨迹残差；
- 跨采集轮次 `\theta` 一致性；
- 跨实验人员 `\theta` 一致性；
- 留出位置/轨迹的预测误差；
- 异常梯度和优化失败率。
- 各误差控制组对应的残差预算和置信区间。

### 13.6 防止错误结论

- 不能只报告训练残差；
- 不能只与固定距离 + 普通 MSE 比较；
- 不能把前置深度输出当无误差真值；
- 不能混合手机上行和 AP 下行 CSI；
- 不能把 CSI 采集网卡输出的数据称为普通路由器端 CSI；
- 不能把 LOS 成功解释为反射多径已经验证；
- 不能用同一人员同一轨迹的数据同时完成标定和测试。
- 不能把降低 `w_geo` 表述为消除了所有人体误差。
- 不能把硬件漂移或场景建模误差错误归因于人体几何。

## 16. 风险与应对

### 16.1 前摄看不到胸腔

应对：先使用人脸三维锚点，通过个体标定换算胸腔；扩大误差协方差；必要时增加肩部关键点或外部深度真值。

### 16.2 头部与躯干不是刚体

应对：比较固定偏移、个体线性和姿态条件模型；将头部转动实验纳入标定；不能把 `face -> chest` 固定变换当绝对真值。

### 16.3 双摄并发不支持原始深度图

应对：优先使用 ARKit 人脸锚点；只有锚点精度不足时才验证 AVFoundation MultiCam；不要假设 ARKit 世界跟踪时一定能读取完整前置原始深度图。

### 16.4 视觉几何准确但对 CSI 改善很弱

应对：检查传播模型是否包含人体影响路径；比较遮挡和非遮挡位置；若科学效应确实较弱，将贡献转为误差边界或鲁棒域随机化，不强行做联合优化。

### 16.5 CSI 链路错误

应对：CSI 采集网卡只处理可明确识别的手机通信帧；记录采集网卡天线坐标以及它是通信目标还是旁路监听设备；论文始终将测量链路写为 `phone -> CSI acquisition NIC`，不写成普通路由器端 CSI。

### 16.6 相位不可用

应对：先使用幅度；记录相位校准失败边界；不能用未经校准的相位支撑结论。

### 16.7 `\theta` 与人体参数不可辨识

应对：用支架固定的无人数据锚定 `\theta`；第一轮只优化 1–2 个参数；执行有限差分梯度、合成真值恢复和 Jacobian 条件检查；分阶段优化；使用前置深度强约束人体几何；在留出轨迹验证。

### 16.8 WiTwin 反射路径限制

当前已验证 WiTwin Core 0.0.2、WiTwin Channel 0.1.0、RayD 0.4.0、DrJit 1.3.1、PyTorch 2.10.0+cu128 下的确定性信道、LOS、CIR 和 CFR。

反射路径 `max_bounces > 0` 尚未在当前组合中通过验证。应对：

- 先做 LOS 和简化传播的科学假设验证；
- 单独建立反射路径兼容性测试；
- 未通过前不声称完整人体反射/多径可微射线追踪已验证；
- 如需更换工具或版本，先做兼容性评估和回归验证，不能直接升级 Channel、RayD 或 DrJit。

### 16.9 `w_human` 被错误并入高斯噪声

应对：主实验先固定人员、衣物和主要姿态；再在固定 `\mathbf r_t` 的条件下改变人员和衣物；单独报告剩余结构化残差，不把前置深度不能处理的人体属性误差归为随机噪声。

### 16.10 手机附近人体传播模型失真

手机与人体距离较近，简化的几何光学人体模型可能无法准确描述天线方向图改变、人体近场耦合和复杂散射。应对：使用固定距离受控 CSI 测量验证模型趋势；先限定结论为当前频率、天线和人体近似条件；若模型只能解释遮挡而不能解释散射，应在论文中明确边界。

### 16.11 “低成本”缺少量化依据

应对：保存完整 BOM、已有设备与新增设备成本、部署时间和采集效率；与能够完成相近任务的 USRP/SDR 配置比较。若没有可比基线，只写“基于商用手机和网卡”，不作强低成本结论。

## 17. 建议的论文主结果结构

如果各阶段通过，论文主结果建议组织为：

1. **现象图**：相同手机位置、不同人体几何下 CSI 幅度/相位变化；
2. **偏差图**：固定 `d_0` 导致的 `\theta` 偏差；
3. **传感器图**：前摄人体几何与外部真值对比；
4. **算法主表**：FIXED、RAW、KF、RTS、MARG、JOINT；
5. **三维消融**：标量 `d_t` 与三维 `\mathbf r_t`；
6. **不确定度消融**：单点输入与协方差边缘化；
7. **真实闭环图**：测试轨迹残差和跨轮次 `\theta` 一致性；
8. **误差预算图**：`w_geo`、`w_human`、`w_hw` 和 `w_sim` 的受控对照；
9. **成本表**：手机/网卡方案与 USRP/SDR 路线的设备和部署对照；
10. **失败边界**：前摄丢失、相位不可用、非遮挡位置、传播模型失真或极弱人体效应。

论文贡献表述必须与证据对应：

- 只有仿真结果时，不能声称真实系统有效；
- 只有固定距离实验时，不能声称自然运动有效；
- 只有训练残差下降时，不能声称环境参数更准确；
- 只有人脸距离时，应写“经标定估计胸腔参考点”，不能写“直接测得胸腔距离”；
- 只有 LOS 验证时，不能声称完整多径反射有效。
