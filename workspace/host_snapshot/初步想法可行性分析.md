# 初步想法可行性分析：面向手机采集 CSI 的鲁棒优化方案

## 1. 分析目标

本文基于 `问题描述.md` 中提出的初步想法，讨论以下方案的可行性：

1. 使用卡尔曼滤波器。
2. 获取人体与手机相对距离误差 `w_{hd}` 的先验。
3. 修改损失函数，将幅度误差与相位误差分开建模。
4. 设置异常判决机制，剔除人体遮挡或异常测量点。
5. 引入先验辅助可微射线追踪的梯度下降。

其中，卡尔曼滤波是最值得重点讨论的部分。它并不能直接“消除人体误差”，但可以在多传感器融合、轨迹平滑、人体-手机相对状态估计、CSI 时间序列去噪等环节发挥作用。

## 2. 问题中的误差结构

原问题中，测量值可写为：

```math
s(loc, h = 1) = x(h = 1) + w
```

其中：

```math
w = w_{gauss} + w_h
```

进一步地：

```math
w_h = w_{hd} + w_{hh}
```

这里：

- `w_{gauss}`：常规测量噪声，通常可以近似为零均值随机噪声。
- `w_{hd}`：人体与手机相对距离、姿态变化带来的误差。
- `w_{hh}`：真实人体与仿真人体模型在体型、衣物、介电参数等方面的差异。

这三类误差的可处理性不同：

| 误差项 | 是否适合卡尔曼滤波 | 原因 |
|---|---:|---|
| `w_{gauss}` | 适合 | 近似随机噪声，满足滤波器的基本假设 |
| `w_{hd}` | 部分适合 | 若可建立人体-手机相对运动模型，则可以估计其低频变化 |
| `w_{hh}` | 不太适合 | 更像系统性建模误差，不是单纯时间序列噪声 |

因此，卡尔曼滤波的合理定位是：降低随机噪声、平滑轨迹、估计慢变隐变量，并为后续可微射线追踪提供更稳定的输入，而不是直接修正所有人体误差。

## 3. 卡尔曼滤波方法的可行性

### 3.1 卡尔曼滤波可以放在哪些位置

在该问题中，卡尔曼滤波至少有四种可能用法：

1. **手机轨迹滤波**：融合 SLAM 与 IMU，得到更稳定的 `loc`。
2. **CSI 时间序列滤波**：对每个子载波的幅度和相位进行平滑。
3. **人体-手机相对状态估计**：估计人体与手机之间的相对距离、方向或姿态偏移。
4. **联合状态估计**：将 `loc`、速度、姿态、CSI 偏差项一起纳入状态空间。

从可行性和收益看，推荐优先级为：

```text
手机轨迹滤波 > 人体-手机相对状态估计 > CSI 时间序列滤波 > 联合大状态滤波
```

原因是：手机轨迹和 IMU/SLAM 的物理关系最明确，滤波器最容易稳定工作；而直接对 CSI 做滤波容易把真实的多径变化也当成噪声抹掉。

### 3.2 方案 A：对 SLAM + IMU 的手机轨迹进行滤波

这是最推荐的第一步。

#### 状态定义

可以将手机状态定义为：

```math
X_t =
\begin{bmatrix}
p_t \\
v_t \\
q_t \\
b^a_t \\
b^\omega_t
\end{bmatrix}
```

其中：

- `p_t`：手机三维位置。
- `v_t`：手机速度。
- `q_t`：手机姿态四元数。
- `b^a_t`：加速度计 bias。
- `b^\omega_t`：陀螺仪 bias。

IMU 用于预测状态，SLAM 输出的位置和姿态用于更新状态。

#### 状态预测

IMU 预测大致为：

```math
p_{t+1} = p_t + v_t \Delta t + \frac{1}{2} R(q_t)(a_t - b^a_t)\Delta t^2
```

```math
v_{t+1} = v_t + R(q_t)(a_t - b^a_t)\Delta t
```

```math
q_{t+1} = q_t \otimes Exp((\omega_t - b^\omega_t)\Delta t)
```

SLAM 的观测为：

```math
z_t =
\begin{bmatrix}
p^{SLAM}_t \\
q^{SLAM}_t
\end{bmatrix}
```

#### 推荐滤波器

由于姿态和运动模型是非线性的，不建议使用最基础的线性 Kalman Filter，而应使用：

- **EKF**：扩展卡尔曼滤波，工程实现相对简单。
- **UKF**：无迹卡尔曼滤波，非线性较强时比 EKF 更稳定，但计算量更大。
- **因子图 / 滑窗优化**：更适合高精度视觉惯性融合，但实现成本更高。

如果研究重点不是 SLAM 本身，建议先用 EKF 或直接使用成熟 VIO/SLAM 系统输出的平滑轨迹。

#### 可行性评价

该方案高度可行，且收益明确。

优点：

- 可以减少 `loc` 的抖动。
- 可以降低由于位置误差导致的 `s` 与 `\hat{s}` 不匹配。
- 不直接依赖 CSI 模型，工程风险较低。

局限：

- 它只能修正手机位置和姿态误差，不能消除人体材质、遮挡等误差。
- 若 SLAM 本身出现漂移或丢踪，滤波器只能缓解，不能从根本上恢复正确轨迹。

### 3.3 方案 B：对 CSI 幅度和相位进行时间序列滤波

这是一个可尝试但需要谨慎的方案。

#### 观测对象

对于每个时间戳和子载波，CSI 可以写成复数：

```math
s_t = A_t e^{j\phi_t}
```

可以分别对幅度和相位建模：

```math
y_t =
\begin{bmatrix}
A_t \\
\phi_t
\end{bmatrix}
```

也可以使用实部和虚部：

```math
y_t =
\begin{bmatrix}
Re(s_t) \\
Im(s_t)
\end{bmatrix}
```

通常更推荐使用幅度和解缠后的相位，因为它们更符合传播模型中的物理解释。

#### 简单状态模型

若假设短时间内 CSI 变化平滑，可以定义：

```math
X_t =
\begin{bmatrix}
A_t \\
\dot{A}_t \\
\phi_t \\
\dot{\phi}_t
\end{bmatrix}
```

采用常速度模型：

```math
X_{t+1} =
\begin{bmatrix}
1 & \Delta t & 0 & 0 \\
0 & 1 & 0 & 0 \\
0 & 0 & 1 & \Delta t \\
0 & 0 & 0 & 1
\end{bmatrix}
X_t + q_t
```

观测为：

```math
z_t =
\begin{bmatrix}
A^{meas}_t \\
\phi^{meas}_t
\end{bmatrix}
```

#### 关键问题

CSI 的变化不一定是噪声。人移动、手机移动、多径快速变化都会造成真实信道变化。如果滤波过强，可能会把有用的物理变化也抹掉。

因此，CSI 滤波更适合以下目的：

1. 去除高频随机抖动。
2. 平滑明显不合理的跳变。
3. 为异常检测提供更稳定的基线。

不适合直接把滤波后的 CSI 当作“更真实”的 CSI，并强行用于更新环境参数 `\theta`。

#### 可行性评价

该方案中等可行。

优点：

- 实现简单。
- 对随机噪声和硬件抖动有帮助。
- 可以为后续残差检测提供平滑基准。

风险：

- 可能削弱真实多径变化。
- 相位需要先做 unwrap 和硬件相位校准，否则滤波结果意义有限。
- 多子载波、多天线情况下状态维度很高，需要逐子载波滤波或降维后滤波。

建议使用方式：

- 对幅度使用轻量滤波。
- 对相位先做校准和 unwrap，再使用较弱滤波。
- 不直接滤除大幅度衰减点，而是把它们交给异常判决或鲁棒损失处理。

### 3.4 方案 C：估计人体-手机相对距离误差 `w_{hd}`

这是卡尔曼滤波最有研究价值的方向。

原问题中，仿真人体与手机之间可能设定为固定距离，例如 40 cm。但真实采集中，人体与手机之间的相对距离和姿态会波动，进而引入 `w_{hd}`。

可以将这个误差转化为一个隐状态估计问题。

#### 状态定义

可以定义人体相对于手机的状态：

```math
X^h_t =
\begin{bmatrix}
d_t \\
\dot{d}_t \\
\alpha_t \\
\dot{\alpha}_t
\end{bmatrix}
```

其中：

- `d_t`：人体中心或胸腔近似点到手机的距离。
- `\dot{d}_t`：该距离的变化率。
- `\alpha_t`：人体相对手机的方向角或姿态角。
- `\dot{\alpha}_t`：方向角变化率。

更简单的版本可以只估计：

```math
X^h_t =
\begin{bmatrix}
d_t \\
\dot{d}_t
\end{bmatrix}
```

#### 状态转移

可以使用常速度模型：

```math
d_{t+1} = d_t + \dot{d}_t \Delta t + q_d
```

```math
\dot{d}_{t+1} = \dot{d}_t + q_{\dot{d}}
```

其中 `q_d` 和 `q_{\dot{d}}` 表示人体手持姿态的随机扰动。

#### 可能的观测来源

人体-手机相对距离并不容易直接测得，但可以从以下数据中构造弱观测：

1. **IMU 观测**

   手机姿态变化可以间接反映手臂摆动。若假设人手持稳定器，手机与人体之间的距离变化较小，IMU 更适合作为姿态稳定性的观测，而不是直接估计绝对距离。

2. **视频观测**

   如果前置或其他视角能看到人体局部，可以用视觉估计手机到人体的相对位置。但常规后置拍摄室内环境时，视频通常看不到人体，因此可用性有限。

3. **CSI 观测**

   人体近场反射或遮挡会影响 CSI 幅度和相位。可以用 CSI 残差作为观测：

   ```math
   r_t = s_t - F_\theta(loc_t, h = 1, d_t)
   ```

   但这里存在循环依赖：要估计 `d_t` 需要较好的 `\theta`，要优化 `\theta` 又希望先知道 `d_t`。因此更适合采用交替优化。

4. **声学或回音观测**

   若手机可以发出或接收声学信号，回音分析可能估计手机与人体或周围反射面的距离变化。这是一个有潜力但工程复杂度较高的方向。

#### 交替优化框架

推荐将 `d_t` 作为人体扰动隐变量，与环境参数 `\theta` 交替估计：

```text
Step 1: 固定人体距离先验 d_t = d_0，优化环境参数 theta
Step 2: 固定 theta，用 EKF/UKF 估计 d_t 的时间序列
Step 3: 固定或弱约束 d_t，重新优化 theta
Step 4: 重复 Step 2 和 Step 3，直到残差收敛
```

对应损失可以写为：

```math
L =
\sum_t \rho(s_t - F_\theta(loc_t, h = 1, d_t))
+ \lambda_d \sum_t (d_t - d_0)^2
+ \lambda_s \sum_t (d_t - d_{t-1})^2
```

其中：

- `\rho` 是鲁棒损失，例如 Huber 或 Cauchy。
- `d_0` 是默认人体-手机距离，例如 40 cm。
- `\lambda_d` 控制距离不能偏离先验太远。
- `\lambda_s` 控制距离随时间平滑变化。

这个形式本质上已经接近“卡尔曼滤波 + 正则化优化”的统一表达。

#### 可行性评价

该方案有较强研究价值，但需要谨慎验证。

优点：

- 直接针对 `w_{hd}` 建模，比单纯滤 CSI 更有物理意义。
- 可以把人体距离波动从环境参数 `\theta` 中分离出来。
- 有机会降低梯度下降对人体误差的错误拟合。

难点：

- `d_t` 缺乏强观测，容易不可辨识。
- 若只依赖 CSI 残差估计 `d_t`，可能和 `\theta`、人体介电参数混淆。
- 需要可微射线追踪支持人体模型参数化，例如人体相对手机的距离、方向、姿态。

建议：

- 初期只估计一维距离 `d_t`，不要同时估计过多人体参数。
- 给 `d_t` 强先验，例如限制在 `[0.25 m, 0.70 m]`。
- 先在仿真数据上验证：人为加入距离扰动，检查滤波器能否恢复。
- 如果仿真中都恢复不了，真实实验中大概率也不可行。

### 3.5 方案 D：联合估计 CSI 偏差项

也可以不直接估计人体距离，而是估计一个慢变的 CSI bias：

```math
s_t = F_\theta(loc_t, h = 1) + b_t + \epsilon_t
```

其中 `b_t` 表示人体导致的慢变偏差。

定义状态：

```math
X_t =
\begin{bmatrix}
b_t \\
\dot{b}_t
\end{bmatrix}
```

然后用卡尔曼滤波估计 `b_t`。

这个方法工程上更简单，但物理解释较弱。它适合作为 baseline，但不适合作为最终核心贡献。原因是 `b_t` 可能会吸收环境参数误差，导致 `\theta` 的优化变得不可靠。

## 4. 获取 `w_{hd}` 先验的可行性

`w_{hd}` 的核心是人体与手机相对几何关系的不确定性。与其完全依赖滤波器，不如先设计可用的先验。

### 4.1 IMU 估计

IMU 可以提供手机姿态、加速度和短时间运动趋势。它对估计 `w_{hd}` 的帮助主要体现在：

1. 判断手机是否稳定。
2. 判断手持姿态是否发生明显变化。
3. 估计手机朝向与人体方向之间的间接关系。

但 IMU 很难单独给出人体与手机的绝对距离。因此，IMU 更适合作为 `d_t` 的平滑约束或异常检测信号，而不是直接观测。

可行做法：

- 当 IMU 显示手机姿态变化剧烈时，降低该时间段 CSI 的权重。
- 当加速度或角速度超过阈值时，认为人体-手机距离可能发生较大变化。
- 将 IMU 稳定性指标加入损失权重：

```math
L = \sum_t \omega^{imu}_t \rho(s_t - \hat{s}_t)
```

其中 `\omega^{imu}_t` 在手机剧烈运动时变小。

### 4.2 回音分析

回音分析理论上可以提供距离相关信息，但实际可行性取决于手机硬件、声学环境和采样同步能力。

优点：

- 距离测量与无线 CSI 相对独立，可以降低不可辨识问题。
- 若能稳定估计手机到人体或附近物体的距离，将显著增强 `d_t` 估计。

困难：

- 室内声学多径复杂，人体反射不一定容易分离。
- 手机扬声器和麦克风的采样率、同步精度、频响都会影响结果。
- 回音路径可能来自墙面、桌面，而不是人体。

因此，回音分析更适合作为拓展实验，而不是主线方案。

### 4.3 实验标定先验

这是最稳妥的方式。

可以先做小规模标定实验：

1. 让实验人员以正常方式手持手机或稳定器。
2. 使用简单外部测量方式记录人体与手机距离，例如人工测量、摄像头侧拍、深度相机。
3. 得到 `d_t` 的统计范围，例如均值、方差、变化速度。

然后在主实验中使用这些统计先验：

```math
d_t \sim \mathcal{N}(d_0, \sigma_d^2)
```

或者设置硬约束：

```math
d_{min} \le d_t \le d_{max}
```

这比试图从 CSI 中完全反推 `d_t` 更可靠。

## 5. 修改损失函数的可行性

原始损失：

```math
L = E[(s - \hat{s})^2]
```

对于复数 CSI，直接使用平方误差可能不够稳健。建议拆成幅度和相位两部分。

### 5.1 幅度-相位分离

设：

```math
s = A e^{j\phi}, \quad \hat{s} = \hat{A} e^{j\hat{\phi}}
```

则损失可以写为：

```math
L =
\lambda_A \rho_A(A - \hat{A})
+ \lambda_\phi \rho_\phi(wrap(\phi - \hat{\phi}))
```

其中 `wrap` 表示将相位差约束到 `[-\pi, \pi]`。

如果使用 dB 幅度：

```math
L =
\lambda_A \rho_A(A_{dB} - \hat{A}_{dB})
+ \lambda_\phi \rho_\phi(wrap(\phi - \hat{\phi}))
```

这通常比直接比较线性幅度更稳定。

### 5.2 可行性评价

该方案高度可行，建议作为默认方案。

优点：

- 幅度和相位的物理意义不同，分开建模更合理。
- 可分别设置权重，避免某一项主导梯度。
- 便于引入鲁棒损失。

风险：

- 相位必须经过校准和 unwrap，否则相位损失会不稳定。
- 若硬件存在公共相位偏移，需要先消除或建模该偏移。

建议：

- 初期可以先只优化幅度，验证流程是否稳定。
- 然后加入相位损失，并使用较小的相位权重。
- 最后再调节 `\lambda_A` 和 `\lambda_\phi`。

## 6. 异常判决机制的可行性

人体遮挡会造成明显的幅度衰减，尤其是直射路径被遮挡时。异常判决机制的目标是避免这些点对 `\theta` 更新产生过大影响。

### 6.1 基于几何的判决

如果已知 AP 位置、手机位置和人体近似位置，可以判断人体是否遮挡直射路径。

例如，计算人体模型到 AP-手机连线的距离：

```math
d_{\perp} = distance(body, line(AP, phone))
```

若：

```math
d_{\perp} < r_{body}
```

则认为直射路径可能被遮挡，降低该样本权重。

### 6.2 基于幅度门限的判决

如果某一时间点 CSI 幅度出现异常衰减，可以判为异常点：

```math
A_t < median(A) - k \cdot MAD(A)
```

其中 `MAD` 是 median absolute deviation。

也可以基于残差判决：

```math
r_t = A_t - \hat{A}_t
```

若：

```math
|r_t - median(r)| > k \cdot MAD(r)
```

则降低该点权重。

### 6.3 可行性评价

该方案高度可行，但建议使用“降权”而不是“删除”。

原因是：异常点并不一定完全无用，它们可能包含人体遮挡或环境几何的信息。直接删除可能损失信息；降权更稳健。

推荐权重形式：

```math
\omega_t =
\frac{1}{1 + \exp(\gamma(|r_t| - \tau))}
```

最终损失：

```math
L = \sum_t \omega_t \rho(s_t - \hat{s}_t)
```

## 7. 引入先验辅助梯度下降的可行性

可微射线追踪的优化变量 `\theta` 往往包含墙面、地面、家具等物体的电磁参数。如果直接从随机初始化开始优化，容易陷入局部最优，或用环境参数错误吸收人体误差。

### 7.1 先验类型

可以引入以下先验：

1. **材料先验**：墙面、玻璃、木材、金属等介电常数范围。
2. **人体先验**：人体与手机距离范围、人体介电参数范围。
3. **几何先验**：SLAM 得到的房间几何结构。
4. **路径先验**：直射路径、一次反射路径的重要性高于高阶反射路径。
5. **平滑先验**：相邻时间戳的人体隐变量或 CSI bias 不应剧烈变化。

### 7.2 推荐优化流程

```text
Step 1: 使用 SLAM + IMU 得到平滑手机轨迹
Step 2: 使用材料表或经验值初始化 theta
Step 3: 固定 theta，仿真 h = 1 下的初始 CSI
Step 4: 计算测量 CSI 与仿真 CSI 的幅度/相位残差
Step 5: 使用异常判决生成样本权重
Step 6: 使用鲁棒幅度-相位损失优化 theta
Step 7: 可选：交替估计人体距离 d_t 与环境参数 theta
```

### 7.3 可行性评价

该方案高度可行，而且应作为整体算法框架的一部分。

关键是不要让人体相关参数过度自由。否则优化器可能用人体参数解释所有误差，导致环境参数 `\theta` 无法被正确估计。

## 8. 推荐整体方案

综合可行性和研究价值，建议将方案组织为三层：

### 8.1 第一层：稳定输入

目标是得到可靠的 `loc` 和相对干净的 CSI。

推荐做法：

1. 使用 EKF/UKF 融合 SLAM 与 IMU，得到平滑手机轨迹。
2. 对 CSI 幅度做轻量平滑。
3. 对相位做校准、unwrap，再决定是否使用弱滤波。

### 8.2 第二层：鲁棒优化

目标是避免人体误差主导 `\theta` 更新。

推荐做法：

1. 使用幅度-相位分离损失。
2. 对幅度和相位分别使用 Huber 或 Cauchy 损失。
3. 使用几何遮挡判决和残差判决，对异常样本降权。

### 8.3 第三层：人体隐变量建模

目标是专门处理 `w_{hd}`。

推荐做法：

1. 初期只估计人体-手机距离 `d_t`。
2. 使用强先验限制 `d_t` 的范围。
3. 使用卡尔曼滤波或平滑正则约束 `d_t` 的时间连续性。
4. 与环境参数 `\theta` 交替优化。

## 9. 最小可行实验设计

为了验证这些想法，建议从最小可行实验开始。

### 9.1 仿真实验

先在可控仿真中验证算法：

1. 固定真实环境参数 `\theta^*`。
2. 人为设置人体-手机距离扰动 `d_t`。
3. 生成带噪声的 CSI。
4. 分别测试以下算法：
   - 无滤波、普通 MSE。
   - SLAM/IMU 轨迹滤波 + 普通 MSE。
   - 幅度-相位鲁棒损失。
   - 异常判决降权。
   - 卡尔曼估计 `d_t` + 交替优化 `\theta`。

评价指标：

```math
Error_\theta = ||\theta - \theta^*||
```

```math
Error_s = ||s - \hat{s}||
```

```math
Error_d = ||d_t - \hat{d}_t||
```

### 9.2 真实实验

真实实验可以分三组：

1. **无人基准组**：尽可能在无人干扰下采集 CSI，用于估计环境参数上限性能。
2. **固定手持组**：人尽量保持固定姿态，测试人体模型误差。
3. **自然手持组**：按真实使用方式采集，测试 `w_{hd}` 影响。

如果算法有效，应看到：

- 相比普通 MSE，鲁棒损失和异常降权能降低 `\theta` 的不稳定更新。
- 相比无滤波轨迹，SLAM/IMU 滤波能降低 CSI-位置配准误差。
- 若 `d_t` 可辨识，卡尔曼估计人体距离应降低残差并提升泛化性能。

## 10. 结论

卡尔曼滤波在该问题中是可行的，但需要放在正确位置：

1. **最稳妥的用途**是融合 SLAM 与 IMU，得到更可靠的手机轨迹。
2. **最有研究价值的用途**是估计人体-手机相对距离 `d_t`，从而减小 `w_{hd}`。
3. **最需谨慎的用途**是直接滤波 CSI，因为真实多径变化可能被误认为噪声。

整体上，建议不要将卡尔曼滤波作为单独解决方案，而是将其嵌入一个更完整的鲁棒优化框架：

```text
SLAM/IMU 滤波轨迹
    + CSI 幅度/相位鲁棒损失
    + 异常点降权
    + 人体距离隐变量的卡尔曼估计
    + 可微射线追踪交替优化
```

这样既能保留卡尔曼滤波的工程可行性，也能将其与可微射线追踪的参数优化目标真正连接起来。
