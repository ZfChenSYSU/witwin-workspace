# 一种使用手机采集 CSI 并优化室内无线可微射线追踪的新方法

## 1. 问题背景

现有室内信道建模主要包括以下几类方案：

1. **统计复高斯建模**：方法简单、计算量小，但建模精度较低。
2. **射线追踪**：具有明确的物理意义，能够进行较真实的传播仿真。
3. **纯数据驱动建模**：依赖数据学习信道特征，但模型通常偏黑盒。
4. **物理建模与数据驱动结合**：例如 2023 年发表在 MobiCom 上的 NeRF2。

第 4 类方法由于兼具可解释性和较高精度，已经成为当前相关研究的重要方向。然而，现有数据采集方式通常依赖高精度 USRP 和定点采集方案，成本较高，商业可用性受限。

本文希望提出一种新的低成本采集与优化方法：使用手机采集多模态数据，并利用这些数据优化室内无线环境中的可微射线追踪结果。

## 2. 场景与数据采集方式

假设研究对象是一个电磁环境基本不变的室内空间。

采集过程中，实验人员手持稳定器在室内移动，并同步采集以下三类数据：

1. **CSI 数据**：通过网卡监听手机与路由器之间的局域网通信，获得路由器端的 CSI。
2. **视频数据**：手机以较高采样率拍摄室内视频。
3. **IMU 数据**：手机同步记录当前 IMU 信息。

由此可以获得同一时间戳下的 CSI、视频和 IMU 三类数据。理想情况下，整体数据处理流程如下：

```text
video + imuData
    -- SLAM -->
3D position of phone + 3D model of the room
    -- ray tracing -->
wireless digital twin of the room at the AP frequency

position + CSI
    -- differentiable ray tracing -->
update parameters of the 3D model
```

## 3. 人体对 CSI 测量的影响

由于采集过程需要人手持手机，人体会对测量得到的 CSI 产生干扰，主要包括：

1. 对无线信号的吸收和反射。
2. 对部分反射体或传播路径的遮挡。

这些影响可能严重改变测量得到的 CSI，使其与 3D 建模和射线追踪仿真得到的结果产生本质差异，进而影响无线数字孪生参数的更新。

因此，在建模和优化过程中，需要显式考虑人体存在带来的误差。

## 4. 可微射线追踪建模

以下以单一频率，例如 2.4 GHz，为例进行建模。

### 4.1 变量定义

- `loc \in R^3`：手机所在的三维坐标。
- `s \in C^1`：该三维坐标对应的信道响应，包含幅度与相位。
- `F_\theta`：可微射线追踪过程，将 `loc` 映射到 `s`。其中 `\theta` 表示环境参数，例如物体介电常数等。本文假设室内环境基本不变，因此 `\theta` 主要由环境决定。
- `h \in {0, 1}`：表示该位置是否存在人体。`h = 1` 表示有人，`h = 0` 表示无人。

### 4.2 仿真与测量

测量数据可以表示为一组：

```math
(loc, s)
```

仿真估计结果记为：

```math
\hat{s}
```

对于射线追踪过程，有：

```math
\hat{s}(loc, h) = F_\theta(loc, h)
```

由于真实测量过程中人体始终参与采集，为了使测量结果与仿真结果具有可比性，仿真时也应引入人体模型，即令：

```math
h = 1
```

于是：

```math
\hat{s}(loc, h = 1) = F_\theta(loc, h = 1)
```

测量值可以建模为：

```math
s(loc, h = 1) = x(h = 1) + w
```

其中：

```math
w = w_{gauss} + w_h
```

这里，`w` 表示测量噪声，`w_{gauss}` 表示常规高斯噪声，`w_h` 表示由于人体存在导致的真实测量值与仿真值之间的误差。

## 5. 人体误差项分析

人体相关误差 `w_h` 可以进一步拆分为以下两类。

### 5.1 距离误差 `w_{hd}`

在仿真中，人体与手机之间通常会设定一个固定距离，例如 40 cm。

但在真实采集场景中，人手持手机时，人体与手机之间的距离并不可控，可能存在分米级波动。由于仿真模型无法获得这一真实距离信息，因此会引入距离误差。

### 5.2 人体模型误差 `w_{hh}`

在仿真中，人体模型通常会设置相对确定的介电常数等参数。

但在真实场景中，不同人的身材、衣物材质和姿态都会有所不同，这些因素都会导致真实人体与仿真人体模型之间存在差异。该差异同样难以通过仿真直接获得。

目前的一个疑问是：`w_{hh}` 是否相对可控，是否可以近似并入 `w_{gauss}` 中处理。

## 6. 损失函数

基础损失函数可以定义为：

```math
L = E[(s - \hat{s})^2]
```

目标是通过优化模型与测量方案，使测量值 `s` 与仿真估计值 `\hat{s}` 尽可能接近。

## 7. 当前核心问题

当前需要重点解决两个问题：

1. 如何设计优化算法或改进测量方案，使 `s` 与 `\hat{s}` 尽量接近。
2. 如何优化可微射线追踪中的梯度下降过程，使参数更新更加稳定、有效。

## 8. 初步想法

### 8.1 减小 `s` 与 `\hat{s}` 的差距

可考虑以下方向：

1. 使用卡尔曼滤波器，对测量数据或估计结果进行平滑与融合。
2. 尝试获得 `w_{hd}` 的某种先验，例如利用 IMU 估计手机与人体之间的相对姿态，或通过回音分析估计距离变化。

### 8.2 优化梯度下降过程

可考虑以下方向：

1. **修改损失函数**

   将相位误差和幅度误差分开计算，并赋予不同权重。由于相位变化可能对优化过程更敏感，将其单独建模可能有助于提升优化稳定性。

2. **设置异常判决机制**

   由于人体遮挡会显著影响直射路径，若直射路径被遮挡，可能产生较严重的幅度衰减。因此可以设置判决机制，例如基于二维坐标关系或幅度门限，剔除明显异常的测量值。

3. **引入先验辅助优化**

   在梯度下降前，可以先利用已有先验和 `h = 1` 的人体仿真模型获得初始仿真结果，再统一对所有环境参数 `\theta` 进行更新。
